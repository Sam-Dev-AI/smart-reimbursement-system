# 🛡️ Smart Reimbursement System

An AI-powered system designed to streamline and automate corporate reimbursement processes with precision and efficiency.

## 🚀 Overview
The **Smart Reimbursement System** leverages advanced AI to validate receipts, categorize expenses, and manage approval workflows, ensuring a seamless experience for both employees and finance teams.

## ✨ Features
- **AI Receipt Parsing**: Automatic extraction of data from receipts using Gemini 2.5 Flash.
- **Smart Validation**: instant check against corporate policies and budget limits.
- **Seamless Approvals**: Multi-level approval workflows with real-time notifications.
- **Rich Dashboard**: comprehensive overview of expense trends and statuses.

## 🛠️ Tech Stack
- **Core**: JavaScript, HTML
- **Styling**: Vanilla CSS
- **AI**: Google Gemini 2.5 Flash
- **Build Tool**: Vite
