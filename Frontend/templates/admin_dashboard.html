Advanced Approval Rules
Configure percentage-based and auto-approval logic.
Percentage-Based Approval

When set below 100%, an expense gets approved even if not all approvers agree. For example, 60% means 3 out of 5 approvers need to approve.

Auto-Approval

When a specific user (like CFO) approves, the expense is auto-approved. If an amount limit is set, auto-approval only works up to that limit.
